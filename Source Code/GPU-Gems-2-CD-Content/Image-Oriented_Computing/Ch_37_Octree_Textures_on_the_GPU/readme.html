Octree Textures on the GPU

GPUGems 2 release

 

What is on the CDROM ?

  • exe - contains the exe files of the runtime demos
  • src - C / C++ code
    • paint - painting tool
    • simul - simulation of liquid flowing along a surface
    • liboctreegpu - octree texture on the GPU library
    • common - common code
    • win32 - third party libraries
  • cg - Cg code
  • appendix - appendix explaining how to get rid of the frac instruction during the tree lookup.
  • picts - demo pictures
  • videos - demo videos, if the runtime cannot run on your system
  • meshes - a few 3ds files

Paiting tool

Launch the demo from the exe directory. Please use the batch file "Paint - RUNME"

Simulation: Liquid flowing along a surface

Launch the demo from the exe directory. Please use the batch file "Simul - RUNME"

Compiling

To compile go into the src directory, launch the VC .NET 7.0 project file "octreetex.sln". Compile in release mode.

The project file expect Cg to be into "C:\Program Files\NVidia Corporation\Cg"

The exe files are copied into the exe directory.
