ABOUT THE SOURCE CODE:

* Created using Visual studio .Net 2003
* Requires NVIDIA CG 1.2 in order to compile the code
* If you find some piece of code interesting for you, feel free of using it in your own projects

ABOUT THE DEMOS:

* Simple demonstrations of the technique presented in GPU GEMS 2 book, not optimized at all
* Both demos require a pixel shaders 2.0 compatible card
* W, S, A, D to move the camera. Use mouse to set camera direction
