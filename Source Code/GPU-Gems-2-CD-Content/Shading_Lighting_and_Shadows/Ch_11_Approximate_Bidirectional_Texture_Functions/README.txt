Approximate BTF Rendering
=========================

You can start the demo program with:

btfViewer.exe material.3pm


Please note, the program requires Cg, glut, and MS Visual Studio
libraries. The program uses GLUI for its user interface. The GLUI
library and the source code (LGPL license) can be found here:
http://www.nigels.com/glt/glui/
